##  Task 3: Heart Disease Prediction

This repository contains my submission for **Task 3: Heart Disease Prediction** as part of the **DevelopersHub AI/ML Internship**.

##  Project Overview

The objective of this project is to build a Machine Learning model that predicts whether a person is at risk of heart disease based on medical attributes. The project covers data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

This project demonstrates a complete supervised machine learning workflow for binary classification using healthcare data.

---

##  Objectives

- Load and explore the heart disease dataset
- Perform data preprocessing and cleaning
- Analyze feature relationships
- Train a classification model
- Evaluate model performance
- Predict the likelihood of heart disease

##  Dataset

The project uses a heart disease dataset containing various medical features, including:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- ST Depression
- Slope of ST Segment
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease)
